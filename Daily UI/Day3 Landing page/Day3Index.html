Daily UI Day 3 - Landing Page 以及 visibility

他的背景是會動的喔!

這禮拜(疑? 的題目是做一個Landing Page，就是一種只有一頁的網頁，我就突發奇想試試看背景是gif的效果會怎麼樣， 剛做完的時候覺得蠻酷的哈哈，不過看久了反而覺得有點太複雜了，也有可能是因為gif的圖示網路上隨便找的資源，不是為產品網頁量身打造的。

不過這次讓我最滿意的地方是做成RWD，就是當螢幕比較小時候，上面的目錄會變成展開式的，就像下面的效果。


要做到這種效果可以用CSS裡的display:none來實現，但用這個屬性有個缺點， 會無法做出有transition屬性的動畫，比如說 .menu { display: none; transition: 0.5s; opacity: 0; } .menu.active { display: block; transition:0.5s; opacity: 1; } 儘管加上了transition和opacity ，他仍然沒有過渡的效果，因為display:none時， 該元素是不存在的，所以transition是沒效果的。

這個時候!
就要用到今天想介紹的visibility屬性了， 他能夠顯示或隱藏元素並且不更改文件的位置和布局，也不會點擊到他，於是我們可以改寫成這樣 .menu { visibility: hidden; transition: 0.5s; opacity: 0; } .menu.active { visibility: visible; transition:0.5s; opacity: 1; } 就可以有過渡的效果了!
