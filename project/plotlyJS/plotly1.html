畫圖表? plotly.js!

前陣子在統計資料的時候需要各種圖表，我就在想有沒有可以快速畫出圖表的東西(除了excel，我不太會用)，結果就被我發現這個! plotly.js!

plotly.js是一個免費的JS函式庫，用法也超簡單，首先我們要先引入函式庫，你可以:
1:直接用 CDN link 引入
<script src="https://cdn.plot.ly/plotly-2.11.1.min.js"></script>
2:或去官網下載js檔直接引入
<script src="plotly-2.11.1.min.js"></script>
然後就可以正式開始了!

要統計資料的話，那我們肯定要先有一筆資料對吧，於是我找來了從107~111年的重要民生物資物價指標，就是看物價漲了多少(好可怕)
先在.js裡打上資料，並且我想以折線圖來做，折線圖在plotly.js裡叫做scatter，於是我們可以寫成這樣 let price = {    x: [107, 108, 109, 110, 111],    y: [103.4, 106.2, 104.9, 107.2, 110.1],    type: 'scatter', }; (注意，每行code都要用逗號隔開喔)
然後要用陣列的方式把資料儲存下來，所以我們這樣打 let data = [price]; 最後再用這個plotly.js規定的函式 Plotly.newPlot('myDiv', data);      // <div id="myDiv"></div>
登愣!最基礎的折線圖非常簡單就可以做出來了! 所以plotly.js真的是很強大的函式庫，完全沒有用到什麼很難的東西，我只知道物價一直漲。


當然，這樣感覺還少了些東西，標題阿，x軸和y軸的名字啊之類的，所以我們可以再這樣
let layout = {    title: '107~110物價指標',    titlefont: 28,    xaxis: {      title: '年(民國)',    },    yaxis: {      title: '指標(以105 = 100 為基準)',    }, } 最後我們再加上一個可以讓表格跟著螢幕改變大小的設定 let config = {    responsive: true, } Plotly.newPlot('myDiv', data, layout, config); 沒錯!這樣就完成一個圖表了!
